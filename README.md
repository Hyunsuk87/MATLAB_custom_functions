# 📌 Project Overview
This repository contains a collection of custom MATLAB functions I developed to streamline my workflow. I created these tools to automate repetitive manual tasks and build a more efficient modeling environment.

## 🚀 Function Descriptions
- Convert_mdl_to_slx : convert Simulink files from .mdl to .slx format.
- Create_Blank_Model : creates a blank Simulink-model.
- Create_Bus : create a Simulink model containing a Subsystem with a dynamically configured Bus Selector and Bus Creator.
- Parameter_Modify : modify parameters in .mat files
